terrably

terrably lets you write Terraform and OpenTofu providers in TypeScript — full type safety, a familiar ecosystem, and a zero-Go build pipeline.

Providers compile into self-contained Node.js Single Executable Application binaries. The Node.js runtime is bundled inside. Operators run terraform apply exactly as they would with any Go provider — no Node.js installation required on their machines.
