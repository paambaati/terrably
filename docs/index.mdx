# terrably

`terrably` lets you write [Terraform](https://developer.hashicorp.com/terraform/plugin) and [OpenTofu](https://opentofu.org) providers in **TypeScript** — full type safety, a familiar ecosystem, and a zero-Go build pipeline.

Providers compile into self-contained [Node.js Single Executable Application](https://nodejs.org/api/single-executable-applications.html) binaries. The Node.js runtime is bundled inside. Operators run `terraform apply` exactly as they would with any Go provider — no Node.js installation required on their machines.

```npm
# Scaffold, implement, build
npx terrably new mycloud
cd terraform-provider-mycloud
npm install
npm run build
# → bin/terraform-provider-mycloud (~130 MB, Node.js runtime embedded)
```
